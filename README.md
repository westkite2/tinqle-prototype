# Features
- Feed screen
- Post details screen
- User profile screen
  
# How to use
1. Clone repository
2. Install node modules
   > npm i
3. Run Expo
   > expo start
4. Install Expo Go app on your device
5. Scan QR code
   * 기기와 컴퓨터가 동일한 Wi-Fi에 연결되어 있어야 함
   * Wi-Fi의 네트워크 프로필은 개인으로 설정
